Headings

Headings from "h1" to "h6" identify blocks and sections. Every page must have one h1. For accessability reasons mostly

Paragraphs

Paragraphs of text go in "p" tags

Lists

Unordered Lists

Unordered lists are marked with bullet points and go in "ul" tags. Each list item goies in an "li" tag.

Types of Lists:

  • Unordered lists ("ul")
  • Ordered Lists ("ol")

Ordered Lists

Ordered lists are numbered and go in "ol" tags. Each list item goes in an "li" tag.

Anchors (i.e. links)

Anchor tags create links to other pages. The URL goes in the "href" attribute.

Go to DuckDuckGo to search for more information

Setting "target="_blank"" will open the link in a new iwndow

MDN Documentation

Anchor tags can also create links to a new location on the same page. Instead of putting a url in the "href" attribute, just use the "id" of the element you want to navigate to. This allows you to Jump to the top of the page

Text Empahsis

Italics

the "em" element adds extra emphasis to the text with the opening and closing tags. This shows up as italic text in most browsers.

Bold

You can also use "strong" element to emphasize text. This shows up as bold text in most browsers.
